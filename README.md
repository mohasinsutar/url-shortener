# Project Title & Description

URL Shortening Sample Applications using node js

### Prerequisites
*node 
*mongod

## Quick Start and Installation

```bash
$ git clone https://github.com/mohasinsutar/url-shortener myapp
$ cd myapp
$ npm install
$ node app
```
